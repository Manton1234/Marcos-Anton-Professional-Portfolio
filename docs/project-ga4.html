UTM Standard

Use consistent UTMs for every campaign: source, medium, campaign, vehicle segment, and listing ID. This lets Arg Tech compare Google Ads, Facebook Marketplace, organic search, referrals, and direct traffic without guessing which channel created the opportunity.

Recommendation

Prioritize high-intent events over vanity traffic. A smaller audience that views inventory, clicks contact, and requests a test drive is more valuable than broad traffic that never engages with a vehicle.
